## 🧠 RAG Prediction Pipeline

```
User Input (place + date + time)
        ↓
[Fetch Context]
  - Historical crowd data (Vector DB)
  - Weather forecast (OpenWeatherMap API)
  - Local news/events (NewsAPI)
  - Day of week / public holiday flag
        ↓
[Embed & Retrieve]  ← vectorStore.ts + retriever.ts
        ↓
[LLM Prompt (LangChain)]  ← predictor.ts
  System: "You are a crowd density analyst..."
  Context: {retrieved docs + weather + news}
  Query: "Predict crowd for [place] at [time] on [date]"
        ↓
[Structured Output]
  { crowdScore: 78, level: "High", peakTime: "9:30 AM",
    bestTime: "7:00 AM", factors: [...], confidence: 0.87 }
        ↓
Display to User
```

### `/api/predict/route.ts` (simplified)

```ts
import { NextRequest, NextResponse } from "next/server";
import { retrieveContext } from "@/lib/rag/retriever";
import { runPrediction } from "@/lib/rag/predictor";
import { getWeather } from "@/lib/api/weatherClient";
import { getRelevantNews } from "@/lib/api/newsClient";

export async function POST(req: NextRequest) {
  const { placeId, date, time, destination } = await req.json();

  const [ragContext, weather, news] = await Promise.all([
    retrieveContext(placeId, date, time),
    getWeather(placeId),
    getRelevantNews(placeId),
  ]);

  const prediction = await runPrediction({ ragContext, weather, news, date, time });

  return NextResponse.json(prediction);
}
```

## 🗄️ Prisma Database Schema

```prisma
// prisma/schema.prisma

model User {
  id        String   @id @default(cuid())
  email     String   @unique
  name      String?
  createdAt DateTime @default(now())
  searches  TravelSearch[]
}

model Place {
  id        String   @id @default(cuid())
  name      String
  type      PlaceType   // STATION | PUBLIC_PLACE | BUS_STOP | AIRPORT
  city      String
  latitude  Float
  longitude Float
  crowdData CrowdRecord[]
}

model CrowdRecord {
  id          String   @id @default(cuid())
  placeId     String
  place       Place    @relation(fields: [placeId], references: [id])
  timestamp   DateTime
  crowdScore  Int      // 0-100
  level       CrowdLevel // LOW | MEDIUM | HIGH | CRITICAL
  source      String   // "historical" | "realtime"
}

model TravelSearch {
  id          String   @id @default(cuid())
  userId      String?
  user        User?    @relation(fields: [userId], references: [id])
  source      String
  destination String
  arrivalTime String
  result      Json
  createdAt   DateTime @default(now())
}

enum PlaceType {
  STATION
  PUBLIC_PLACE
  BUS_STOP
  AIRPORT
  METRO
}

enum CrowdLevel {
  LOW
  MEDIUM
  HIGH
  CRITICAL
}
```

## 🗓️ Travel Planner Logic

### Page: `/travel-planner`

**User Input:**
- Source (e.g., Kalyan)
- Destination (e.g., CSMT)
- Desired Arrival Time (e.g., 10:00 AM)

**Backend Logic (`/api/travel-plan/route.ts`):**

```ts
export async function POST(req: NextRequest) {
  const { source, destination, arrivalTime } = await req.json();

  // 1. Calculate average journey duration (from historical data or static config)
  const avgDuration = await getAvgJourneyTime(source, destination); // e.g., 75 mins

  // 2. Generate prediction for multiple departure windows
  const windows = [-30, -15, 0, +15, +30]; // minutes offset from ideal departure
  const idealDeparture = subtractMinutes(arrivalTime, avgDuration);

  const predictions = await Promise.all(
    windows.map(offset => {
      const departTime = addMinutes(idealDeparture, offset);
      return predictCrowdForWindow(source, destination, departTime);
    })
  );

  // 3. Recommend the window with lowest crowd score
  const recommendation = predictions.sort((a, b) => a.crowdScore - b.crowdScore)[0];

  return NextResponse.json({ recommendation, allWindows: predictions });
}
```

**Result shown to user:**
```
🟢 Best Time to Leave: 8:15 AM
   Expected Crowd at Kalyan Station: LOW (32/100)
   Expected Crowd at CSMT: MEDIUM (54/100)
   Est. Arrival: 9:50 AM ✅ (before 10:00 AM)

⚠️  If you leave at 8:45 AM: HIGH crowd (78/100)
```

## 📡 External API Integration

### Weather API (`/lib/api/weatherClient.ts`)

```ts
export async function getWeather(location: string) {
  const res = await fetch(
    `https://api.openweathermap.org/data/2.5/weather?q=${location}&appid=${process.env.OPENWEATHER_API_KEY}`
  );
  const data = await res.json();
  return {
    condition: data.weather[0].main, // Rain, Clear, Thunderstorm
    temp: data.main.temp,
    humidity: data.main.humidity,
  };
}
```

Weather impacts prediction: Rain → Higher crowd at covered stations; Extreme heat → Lower outdoor crowd.

### News API (`/lib/api/newsClient.ts`)

```ts
export async function getRelevantNews(location: string) {
  const res = await fetch(
    `https://newsapi.org/v2/everything?q=${location}+transport+crowd+event&apiKey=${process.env.NEWS_API_KEY}&pageSize=5`
  );
  const data = await res.json();
  return data.articles.map((a: any) => ({
    title: a.title,
    description: a.description,
    publishedAt: a.publishedAt,
  }));
}
```

News impacts prediction: Strikes, events, marathons, bandhs → Spike in crowd.

## 🔔 Push Notifications Setup

```ts
// hooks/useNotifications.ts
export function useNotifications() {
  const subscribe = async () => {
    const reg = await navigator.serviceWorker.ready;
    const sub = await reg.pushManager.subscribe({
      userVisibleOnly: true,
      applicationServerKey: process.env.NEXT_PUBLIC_VAPID_PUBLIC_KEY,
    });
    await fetch("/api/notifications/subscribe", {
      method: "POST",
      body: JSON.stringify(sub),
    });
  };
  return { subscribe };
}
```

Notifications will fire when:
- Crowd at a saved station reaches HIGH/CRITICAL
- A saved travel route has an upcoming peak window
